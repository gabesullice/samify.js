Samify.js

The flexible jQuery equalizer

View project on GitHub

Responsive, flexible table-like layouts

Samify.js is a dead simple, lightweight jQuery plugin that allows you to create responsive, flexible, table-like layouts. Whether it's equal height, equal width, both or even equalized by each row and column (table-like), samify.js lets you quickly create equalized DOM elements. It's simple:

<div class="box-container">
  <div class="box"><img src="..."/></div>
  <div class="box"><p>...</p></div>
  <div class="box"><p>...</p></div>
  <div class="box"><img src="..." /></div>
  <div class="box"><p>...</p></div>
</div>
$(".box").samify();

Options

The samify() method takes a single options object with the following keys:

height

Default: true This will equalize all elements to the height of the largest sibling.

width

Default: false Same as height, but equalizes widths to the width of the widest element.

vshrink

Default: false This overrides the height option. This will equalize each row to the height of the largest element in each row. E.g. In a grid 3 wide and 2 high, the top row will equalize to the height of the largest element among the first 3 elements. The bottom row will equalize to the maximum height of the last 3 elements.

hshrink

Default: false Same as vshrink, but equalizes by column.

cols

Default: 0 This can be either a number or a function. If a number, the number of columns will always remain fixed. If a function, the number of columns can be dynamic and recalculated on resize. The function takes on parameter, the window width. Note: You must also set resize to true for the function to recalculate on resize, it will be called only once otherwise.

$(".box").samify({
  'resize': true,
  'cols': function (width) {
    ...
  }
});
resize

Default: false Whether or not to bind to window resize and recalculate sizes and columns on resize.

throttle

Default: 250 The delay in ms that the grid will be resized on window resize.
